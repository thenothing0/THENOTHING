# HYDRA Skills for Claude Code
# Each .md file in this directory is a reusable skill/prompt

This directory contains reusable skills that Claude Code can reference.
Place `.md` files here with specific methodologies for different vulnerability types.

## How to use

In Claude Code, reference skills with:
```
Use the blind_xss_hunter skill on https://target.com
```

Or load them directly:
```
@skills/blind_xss_hunter.md
```
